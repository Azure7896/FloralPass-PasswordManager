# FloralPass-PasswordManager

![floralpasslogo](https://github.com/Azure7896/FloralPass-PasswordManager/assets/95986791/83e0dbed-c310-463f-a58b-f90bfb76a0f8)

<p align="center"><b>FloralPass is a simple password manager.</b></p>

<p align="center">⚠️⚠️⚠️Work in progress!⚠️⚠️⚠️</b></p>

Used technologies:

  - Kotlin
  - SQLite

Used applications:

  - Android Studio
  - DBBrowser

Main functions:
  - Register:
      - Only one user per device
      - Inputs validation
  - Login
      - With block input for 30 seconds after typing an incorrect password 3 times
  - Toast notifications
  - Standard notiications
  - Password encrypting/decrypting
